1. 通过路由地址控制权限

    ```
    在程序的入口文件中
    通过的方式 实现一个权限的判断,对某些页面进行是否登陆的权限判断 
    app.all('/admin/*',(req,res,next)=>{
        if(){ /////没有登陆
            res.redirect('')////跳转到登陆页面
        }
        else{
            next()
        }
    })
    ```
